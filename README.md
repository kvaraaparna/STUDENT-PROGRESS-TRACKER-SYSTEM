# STUDENT-PROGRESS-TRACKER-SYSTEM

**##Describtion:**

The Student Progress Tracker System is a Python-based mini project developed to help manage and monitor the academic progress of students in a simple and organized manner. This system allows users to store student information such as name, roll number, and marks, and display progress reports when required.

The language are using to develop the suatem is Python programming concepts like lists, functions, conditional statements, and loops. Instead of learning concepts only through theory, this project helps students apply them in a real-time application.

**##objectives:**
* Manage student information
* Monitor academic progress
* Generate simple progress reports

**##Technologies used in this student progress tracker system is**
* python
* loops
* conditions
* list
* operations


**##How to run the code**
1.Enter the student details like name,class name,roll no, haow many subjects user want to.

2.Enter marks of each subject and click enter.

3.It return the total marks ,average marks,highest marks,lowest marks,marks percentage and grade that the user got in the form of report.

4.After that choose any one the three choices (enter marks again,view report again or exit.
